# Sistem Putusan Narkotika

## Deskripsi Proyek

Aplikasi JavaFX berbasis MVC untuk mengelola data putusan perkara narkotika.

## Fitur

* Menambah data putusan
* Menampilkan seluruh data
* Mencari data berdasarkan nomor perkara
* Validasi input menggunakan Exception Handling
* Statistik jumlah data putusan

## Teknologi

* Java 21+
* JavaFX
* Maven
* Git & GitHub

## Cara Kompilasi

```bash
mvn clean compile
```

## Cara Menjalankan

```bash
mvn javafx:run
```
